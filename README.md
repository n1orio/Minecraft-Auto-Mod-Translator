# Minecraft Auto Mod Translator

Скрипт на Python, который автоматически проверяет все ваши моды на наличие русского языка. Если перевода нет, скрипт вытаскивает английскую локализацию, переводит её с помощью Google Translate, восстанавливает форматирование и собирает всё в один готовый **Ресурспак**

## ✨ Особенности

- **Независимость:** Не нужно класть скрипт в папку с игрой. При запуске появится окно, где вы сможете визуально выбрать вашу папку `mods`.
- **Батч-перевод (Скорость):** Текст переводится "пачками" (по 50 строк). Это работает в 10 раз быстрее построчного перевода и защищает от бана IP Гуглом.
- **Защита форматирования:** Умный алгоритм чинит сломанные переводчиком переменные Minecraft (исправляет `% 1 $ s` на `%1$s`, `§ a` на `§a` и т.д.).
- **Поддержка любых версий:** Умеет парсить как новые моды (`.json`), так и моды для 1.12.2 и ниже (файлы `.lang`).
